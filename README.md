# Digital Thermometer using Arduino

## Project Overview
This project is a Digital Thermometer using an Arduino Uno and a DS18B20 temperature sensor. It measures temperature and displays the value in real time using the Serial Monitor.

## Objective
- Measure temperature accurately.
- Display real-time temperature.
- Learn Arduino sensor interfacing.

## Components Used
- Arduino Uno
- DS18B20 Temperature Sensor
- 4.7kΩ Resistor
- Breadboard
- Jumper Wires
- Wokwi Simulator
- Arduino IDE

## Working Principle
The DS18B20 sensor measures the temperature and sends digital data to the Arduino. The Arduino processes the data and displays the temperature in degrees Celsius.

## Software Used
- Arduino IDE
- Wokwi Simulator
- GitHub

## Results
The project successfully displays real-time temperature readings.

## Applications
